# ASRock-Z370M-ITX-AC
opencore efi for ASRock Z370M ITX/AC 

基于OC 0.6 正式版编写的适用于华擎Z370M ITX/AC主板的efi引导文件

使用前请记得修改三码

bios版本：4.20（当前最新版）
cpu: 9900es qqz5
主板：华擎z370m itx/ac
内存： juhor ddr4 2666 8g*2
显卡：华硕dual rx580 o8g
硬盘：intel 660p 512g
有线网卡：主板自带的有线intel i219v+intel i211
免驱无线卡：bcm94360cs2
